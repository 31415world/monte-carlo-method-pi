# モンテカルロ法で円周率を求める
モンテカルロ法で円周率を求める＆収束をグラフで表示するプログラム.  

## ファイル構成
### 円周率計算
```
index.html : 値入力用、グラフィック表示、ここからプログラムを実行.  
script - pi.js : 実際に実行されるプログラム.  
```
### 収束計算
```
converge.html : 収束を求め、グラフを表示させる用.  
script - converge.js : 実際に実行されるプログラム.
```

## サンプル画像
### 円周率計算
![image](https://user-images.githubusercontent.com/74352831/205846447-5e1fb2d5-d560-43b5-b8c5-c3b0fa526e32.png)  
点の数を指定し、指定された数を用いてプログラムを実行する.  
サンプル画像は点の数10000個で結果は3.1472  
詳しくは→https://3.1415.world/tools/monte-carlo-method-pi/
### 収束計算
![image](https://user-images.githubusercontent.com/74352831/205862318-1b43a6cb-e37f-49f6-af0f-a1c8c3f4aad3.png)

サンプル画像は10000個点がある場合までの収束.
